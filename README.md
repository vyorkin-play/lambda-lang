# Lambda Lang

A simple programming language written in JavaScript (lol).

```
    sqr = def(x) x * x;
    cube = def(x) x * x * x;
    z = 4;
    x = 8;
    y = 5;
    cube(z + x) - sqr(y) + 2
```
